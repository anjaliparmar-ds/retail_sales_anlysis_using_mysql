🛍️ Retail Sales Analysis using SQL

 📌 Project Overview
This project analyzes retail sales data using **MySQL** to uncover meaningful business insights. It answers common business questions related to sales performance, customer behavior, product categories, and purchasing trends using SQL queries.
The project demonstrates practical SQL skills such as data filtering, aggregation, grouping, window functions, and business data analysis.

 🚀 Features
- 📅 Analyze sales on specific dates
- 👕 Filter transactions based on category and quantity
- 💰 Calculate total sales by product category
- 👥 Analyze customer demographics (age & gender)
- 📈 Identify high-value transactions
- 🏆 Find top 5 customers by total sales
- 📊 Calculate unique customers per category
- 📆 Identify the best-selling month of each year
- ⏰ Analyze order distribution by sales shift (Morning, Afternoon, Evening)
- 📑 Solve real-world business problems using SQL

 🛠️ Tech Stack
- Database: MySQL
- Language: SQL
- IDE:MySQL Workbench

📊 Key Business Questions Solved
1. Retrieve sales made on a specific date.
2. Find Clothing transactions with quantity greater than 3 in November 2022.
3. Calculate total sales for each category.
4. Find the average age of customers purchasing Beauty products.
5. Identify transactions with sales greater than ₹1000.
6. Count transactions by gender and category.
7. Determine the best-selling month in each year.
8. Find the top 5 customers based on total sales.
9. Count unique customers for each category.
10. Analyze order distribution by Morning, Afternoon, and Evening shifts.

⭐ Key Highlights
- Applied SQL to solve real-world retail business problems.
- Used aggregate functions (`SUM`, `AVG`, `COUNT`).
- Implemented filtering with `WHERE`, `AND`, `MONTH()`, and `YEAR()`.
- Used `GROUP BY` and `ORDER BY` for data summarization.
- Applied the `RANK()` window function to identify top-performing months.
- Used `CASE` statements for shift-based sales analysis.
- Generated business insights from transactional retail data.

📌 Project Outcome
This project demonstrates strong SQL fundamentals and the ability to transform raw retail data into actionable business insights. 
It showcases practical experience with data analysis techniques commonly used in Data Analyst roles.
